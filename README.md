# TOFTEC-Generator-Frontend

TOFTECT-Generator is "termination of the employment contract" generator in .pdf extension. 

Used technologies:

  - Angular with Material UI

User applications:

  - Webstorm
  - Google Chrome
  
Application is deployment on AWS Amazon S3.
