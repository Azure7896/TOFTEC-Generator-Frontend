# TOFTEC-Generator-Frontend

TOFTECT-Generator is "termination of the employment contract" generator in .pdf extension. 

Used technologies:

  - Java 11
  - Spring Boot 2.7.6 
  - iText 7.2.5

User applications:

  - IntelliJ IDEA
  - Postman
  
Application is deployment on AWS Beanstalk with Tomcat.
